                          MODAL IMPLEMENTATION
================================================================================

JavaScript Features:
  • Event Listeners:
    - Button click: Opens modal with .active class
    - Close button click: Closes modal + stops video
    - Overlay click: Closes modal + stops video
    - Escape key press: Closes modal + stops video

  • Video Control:
    - Stops by resetting iframe src
    - Prevents background playback after close

  • Smooth Transitions:
    - CSS-based animations
    - Timing: 0.2s ease
    - Z-index: 1000 (above all other content)

HTML Structure:
  • Modal container (position: fixed, full viewport)
  • Modal overlay (semi-transparent dark background)
  • Modal content (card with rounded borders)
  • Video container (responsive 16:9 aspect ratio)
  • Close button (circular, top-right)

CSS Classes:
  • .modal (parent, display: none by default)
  • .modal.active (display: flex when visible)
  • .modal-overlay (clickable background)
  • .modal-content (card wrapper)
  • .modal-video-container (aspect ratio keeper)
  • .modal-close (close button styling)

Responsive Behavior:
  • Desktop: 900px max-width, 80vh max-height
  • Mobile: 95% width, 60vh max-height
  • Close button scales appropriately
